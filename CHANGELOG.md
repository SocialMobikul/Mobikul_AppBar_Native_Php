# Changelog

All notable changes to this project will be documented in this file.

This file is the single source of truth for release history.
Any files under `release/*` are optional supplementary artifacts.

## [1.0.0] - 2026-05-14

### Added
- Initial NativePHP plugin release for dynamic app bar configuration.
- Bridge methods:
  - `MobikulAppBar.Configure`
  - `MobikulAppBar.Reset`
- Context-aware icon defaults (`home`, `product`, `checkout`, `profile`, `default`).
- Allowed icon validation for stable payloads.
- Laravel service provider and facade registration.
- Optional HTML/CSS/JS app bar helper for Blade or hybrid web views.
- Release notes and demo app test instructions.
